GoldenCheetah v3.3

We are proud to announce that GoldenCheetah v3.3 was released today.

Version 3.3 is an incremental release and contains a handful of new features plus some fixes for some of the more severe bugs found in version 3.2

MAJOR NEW FEATURES

  1. Sync across PCs via Dropbox cloud storage
  2. Sync via local folder, thumb drive, Google mounted drive etc
  3. Added Daniels VDOT and T-Pace tools for Running
  4. User Formulas
  5. Restful API Web-Services for integration with R,Matlab,Orange,Tableau etc
  6. Support FE-C trainer
  7. Monark ERG Support
  8. Support Stryd Power Meter for Running
  9. BSX Insight 2 support
  10. Ambit SML import support
  11. RLV Support, playback to riding speed.
  12. Video Overlays in Train View
  13. SportsPlusHealth upload support
  14. Backup and Autobackup
  15. Better swim workout support incl. lap workouts
  16. SrmRideFile: v9 support (SRM PC8)
  17. Add new language support: Chinese(Traditional)
  18. GPL v3 License

SEVERE BUG FIXES

  1. Fix 3DP file import 'hang'
  2. Joule GPS+ import issues
  3. Checkbox metadata fields not saved, ^S save errors
  4. Fix GPU battery drain on Mac OSX
  5. Fix more than 8 HR Zones Crash
  6. Fix 'Fix Elevation' SEGV
  7. Fix Interval Search SEGV on High Power
  8. DataProcessor 'Auto' runs on import only
  9. Fix Memory Exhaustion on Mass Import/Sync
  10. Fix SEGV when sharing activity on Strava with no internet
  11. Fix IntervalNavigator SEGV
  12. Fix SEGV LTM Edit

MINOR NEW FEATURES

  1. Delete multiple workouts
  2. Configure CP and FTP separately
  3. FixSmO2: Add a tool to remove anomalies in SmO2 data
  4. Recognise more Garmin devices based on FIT SDK
  5. Fix Freewheeling Tool
  6. Added W'bal Work In Zone Metrics
  7. Add W'bal Zones - Time above CP
  8. Added Autoimport Stealth/Background Mode
  9. TrainMode - Use Multimedia Keys
  10. Fix Speed from Distance tool
  11. Delete Athlete now supported
  12. Settings stored in athlete folder config directory
  13. Bugfix merge of hrm and gpx
  14. Mass update metadata with set, isset and unset commands
  15. Add a Derive Distance tool (from GPS position)
  16. Read R-R data from polar hrm files
  17. Add User parameters for Bike Weight and CRR to Power Estimation Tool
  18. Multiple fixups for DataFilter expressions and precedence logic
  19. LTM Filter for a curve
  20. Rename Route
